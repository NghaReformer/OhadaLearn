# Écritures Comptables Courantes

## 1. Vente au comptant

Vente de marchandises contre espèces (sans TVA pour simplifier) :

| Compte | Débit | Crédit |
|--------|-------|--------|
| `521` Banque | 1 500 000 | |
| `701` Ventes de marchandises | | 1 500 000 |

*L'actif (Banque) augmente → débit. Le produit (Ventes) augmente → crédit.*

## 2. Achat à crédit

Achat de marchandises à crédit :

| Compte | Débit | Crédit |
|--------|-------|--------|
| `601` Achats de marchandises | 800 000 | |
| `401` Fournisseurs | | 800 000 |

*La charge augmente → débit. Le passif (Fournisseurs) augmente → crédit.*

## 3. Règlement fournisseur

| Compte | Débit | Crédit |
|--------|-------|--------|
| `401` Fournisseurs | 800 000 | |
| `521` Banque | | 800 000 |

*Le passif diminue → débit. L'actif (Banque) diminue → crédit.*
